O1: Given array of students marks [23, 45, 67 78,91] add 10 in all the elements of an array by using map and forEach function

         MAP 

         let arr = [23, 45, 67, 78, 91];

        let arr2 = [23, 45, 67, 78, 91];

3 Different ways of creating object


          # Using Object literals

            let obj = {
            first:"firstName",
            last:"lastName"
            }

          # using Object Constructor

            let obj = new Object();
            obj.first = "firstName";
            obj.last = "lastName";

          # using Object.create

            let obj = {}
            let obj1 = Object.create(obj);
            obj1.first = "firstName";
            obj1.last = "lastName";
            console.log(obj1);

call, bind and apply in JS

            let obj = {
                name:"john",
                age:22,
            }
            
            function xyz(salary){
                console.log(this.name,this.age,salary);
            }
            
          # using call
            call method basically allows you to call a function and then pass in a specific value.
            xyz.call(obj,25000);
            
          # using apply
            The apply() method calls a function with a given this value, and arguments provided as an array.
            xyz.apply(obj,[25000]);
            
          # using bind
            Bind method takes an object as an argument and creates a new function.
            let st = xyz.bind(obj,25000)
            st();

Prototype chaining

            Each object has a private property which holds a link to another object called its prototype.
            That prototype object has a prototype of its own, and so on until an object is reached with null as its prototype.
            null has no prototype, and acts as the final link in this prototype chain.

            function Box(height,width) {
                this.height = height;    //   "this" keyword in this function refers to Box object.
                this.width = width;
              }
            
              let box = new Box(10,20);      // "new" operator creates an object { }.
